Assignments for Tuesday, January 27

Design Principles: Visual Perception and the Principles of Gestalt

View Presentation

  • Using Photoshop or Illustrator, create illustrations for each of the primary principles of Gestalt listed below.
    • Similarity
    • Continuation
    • Closure
    • Proximity
    • Figure / Ground
    • Symmetry and Order
  • Think of a real-word web design scenario where each of these principles could be put to use. Write a short description of the situation and how the principle applies, and add it to the appropriate illustration above. Bonus points for real-world examples.
  • Save each example as a JPG file.
  • Attach all six examples to an email and send to your instructor.
