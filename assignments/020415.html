Assignments for Wednesday, February 4

SASS

Download In-Class Examples

Welcome to the world of SASS! Your assignment is to switch your latest project to use SASS.

  1. Create a folder named "scss" to contain your new sass files.
  2. Copy current styles from "styles.css" into a new SASS file named "styles.scss" and save it into the "scss" folder.
  3. Run SASS so that it watches for changes and compiles your CSS: sass --watch scss:css
  4. Use variables, mixins, partials, etc. as appropriate to simplify your CSS workflow.
  5. Email your instructor with a link to the code on GitHub.
