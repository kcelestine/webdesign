Assignments for Tuesday, January 6

Git Setup:

(To review the Git process, see https://try.github.io/

  1. Run git init in the "outlanderspices" directory.
  2. Add a .gitignore file to the directory.
  3. OPTIONAL: Add a README.md file to the directory, with a title and a description.
  4. Run git add to stage all files in this directory, and run git commit -m "message" to commit them.
  5. Create a new repository on your GitHub account called "outlanderspices"
  6. Use the commands from GitHub to link your remote repository to your local files.
  7. Run git push -u origin master to push your files to GitHub.

HTML:

  1. Add a top navigation (using a list of links) to all pages of the Outlander Spices website. Link to the pages we have already created. Assume the following sitemap:
    • Home
    • About Us
    • Products
    • Recipes
    • Order Online
    • Locations
    • Contact Us
  2. Create a new page named, "contactus.html." On this page, create a form to collect user information, with appropriate form fields for the following:
    • Name
    • Address
    • City
    • State
    • Zip
    • Email address
    • Choose your favorite spice (from the list spices on the product page)
    • Share how you have used Outlander Spices to spice up your favorite dish
    • Choose professional chef or home cook
    • Sign me up for your newsletter
    • Submit buttons
  3. On the "About Us" page, create a list of links at the top of the content that link to the "All Spiced Up," "Expansion Project" and "The Project Team" sections of the same page. Create "Back to top" links to return to the top of the page.
  4. Commit your changes and to push them to GitHub. Email the GitHub link to your instructor.

Beast Mode:

  1. Create a page called "orderonline.html" with an appropriate order form. How would you accept orders for a site that sells spices? What type of information would you need to collect?
  2. Create a "Locations" page for Outlander Spices. Where are their stores located? What types of information would the user hope to find on this page? How would that content be marked up in HTML?
