Assignments for Thursday, January 8

Outlander Spices

  1. Complete the Outlander Spices homepage.
    1. Include a page header, with an h1 containing the title, "Outlander Spices." Replace the text with "logo.jpg" as a background image.
    2. Add "headerspices.jpg" to the top of the page as a background, and position it as seen in the design above.
    3. Create a main marketing area using "homespices.jpg" as a background image. Style the headline and the supporting text as seen in the design.
    4. Create a two-column layout on the homepage for "In the News" and "Featured Product" using floats as seen in the design.
    5. Position the images in the "Featured Product" column so that they appear to the left of the text.
    6. Adjust overall spacing so that the HTML site looks as close to the design as possible. Think "pixel-perfect."
  2. Complete the rest of the pages of the site, making sure that each page looks polished when the stylesheet is applied.
  3. Push the completed site to GitHub.

 

 

Citizens Information Center

You've recieved an updated design for the Citizen's Information Center!

  1. Build the HTML and CSS for this website.
  2. Build all four pages: Home, Infomation Center, Newsroom, and Contact Us. Create a single CSS stylesheet for the site and link it to these pages.
  3. Add hover states to the Citizens navigation links, as seen in this example.
  4. On the Information Center page, markup and style the content available at this link. Create a set of same-page jump links and "Back to top" links to make navigating this page easier.
  5. Use the existing content for the Newsroom page. Add the "blueribbon.jpg" and "newsroom.jpg" images to the page, and style it as seen in this design.
  6. Create a form on the Contact Us page that collects the following information: Name, Email, Phone, Sign up for the Newsletter?, Comments. Style the form fields, labels, and submit buttons to resemble the design of the site.
  7. Add a .gitignore file to the citizens directory. (We do this to prevent system files from being tracked by Git.)
  8. Add a README.md file to the citizens directory, with a proper title and description, (We do this so that once your files appear on GitHub, visitors will see this information.)
  9. Create a new GitHub repository named "citizens" and push the completed site.
